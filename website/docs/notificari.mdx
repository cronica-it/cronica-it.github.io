# Notificări (feeds)

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

Pentru cei care doresc să fie la curent cu noile articole postate
pe acest site, RSS feeds sunt disponibile pentru fiecare dintre cele trei
secțiuni (Evenimente, Amintiri, Blog).

Toate cele trei formate comun utilizate (`rss.xml`, `atom.xml` și `feed.json`)
sunt disponibile.

Cea mai comodă metodă de configurare este cu _copy_ în câmpul de mai jos
și _paste_ în programul care integreaza feed-urile de RSS (de exemplu [feedly](https://feedly.com)).
